Work Request 1: We need a report that will divide out monetary donations by Rotary Club. There are 52 Rotary Clubs that make up Rotary District 5030. Rotary First Harvest is a project of this Rotary District.

To track how each club is supporting our work we need a report that shows total donations for these 4 categories:
  • Individuals
  • Individual Gift Cards
  • Club Donations
  • Club Gift Cards

The report also needs to show the total of those 4 categories for each of the 52 clubs, plus work party volunteer hours.

Currently, we have to print out 3 different reports from Salesforce, then manually enter the numbers into an Excel spreadsheet. We would like one report that would pull all of this information at one time.
