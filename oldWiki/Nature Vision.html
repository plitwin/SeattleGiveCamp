Project Proposal


"We are looking for help to improve our Access database which is used to keep track of nearly 2,000 science and nature programs annually. Cindy Potter, a former staff member, set up the database and created a format that she still updates once a year. The current database separates records from each school year into separate tables. As a result of using this format, if we want to search for program statistics from previous years, our registrar has to go into an old Access file that contains the information from a previous school year. This results in complications when we need to find information by calendar year rather than school year for certain funders.

It is our desire to have all database years merged into one file that is easily searchable and usable by our current registrar. There are many shortcuts set up in the current database for her that allow her to easily run common reports without needing to spend more time learning Access/queries etc. While it is our ultimate goal to find the funds to pay our registrar to learn more about Access, at this time we just need a more up-to-date way to gather and store information. It is not practical for us to make a big change in the way we keep and use our info, nor in how the database is set up. However, having all information in one place would make such a big difference for a nonprofit with a small staff and budget. We appreciate your consideration of this project.

Cindy Potter will be around during the weekend to help explain the database and brainstorm solutions. There are several other database improvements that would be nice to have. Many of these are simple changes. Cindy is currently cleaning up the database. She will also be available after the project is over to help train the staff and make sure the reports in the database are working correctly."
